# 행동
이 문서는 체스택에 등장하는 각 행동들을 정리한 문서입니다.
각 행동은 이동 행동을 제외하고 모두 한 턴당 한번만 할 수 있습니다.

## 착수
자신 포켓에 가진 기물 중 하나를 선택해 보드의 원하는 곳에 놓습니다. 단 프로모션 가능한 기물의 경우 프로모션 되는 칸에 착수할 수는 없습니다.

## 이동
보드 위의 자신 기물을 이용해 각 기물들의 행마법에 따라 보드 위에서 특정한 행동을 수행합니다. 
한 턴에 한 기물을 여러 번 움직일 수는 있으나, 한 턴에 여러 기물을 움직일 수는 없습니다.
행마법은 chessembly로 작성됩니다. 자세한 설명은 chessembly/ 폴더의 문서들을 참조하시오.
특정한 행동에 대한 내용은 chessembly/TUTORIAL.md 파일 참조하세요.

## 위장
보드 위의 자신의 로얄 피스중 하나를 선택해 다른 기물로 위장시킵니다. 이때 스택의 부여 규칙은 위장한 기물을 따라가나, 스턴 스택값은 위장 후에도 계승됩니다.

## 계승
보드 위에 아군 기물 하나를 선택하여 로얄 피스로 만듭니다.

## 스턴
보드 위에 기물 하나를 선택해 스턴 스택을 부여합니다. 아군 기물은 1~3스택, 적 기물은 1스택만 가능합니다.
